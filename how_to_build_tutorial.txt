Для сборки тч:

*********************************************************************

Установить Microsoft Visual Studio 8 (2005) + sp1 + sp1 for vista
https://rutracker.org/forum/viewtopic.php?t=2566619
из раздачи нужны:
Visual Studio 2005 Professional English - en_vs_2005_pro_dvd.iso
Service Packs - Visual Studio 2005 SP1 - VS80sp1-KB926601-X86-ENU.exe
Service Packs - Visual Studio 2005 SP1 Update for Windows Vista - VS80sp1-KB932232-X86-ENU.exe

***

Установить Borland C++ Builder 6
https://rutracker.org/forum/viewtopic.php?t=3638325

Исходный код игры: S.T.A.L.K.E.R. Тень Чернобыля (1.0007rc1) (*1)
Скачать: https://drive.google.com/file/d/1GVTzdiMNVhKK-6C831J_GMhfeEVmNyI4/view?usp=sharing

и распаковать архив xray_engine.rar, как пример: D:\xray_engine

Скопировать из D:\xray_engine\sdk\for_borland\, папку Borland в C:\Program Files (x86) или туда, куда установлен борланд.

Скопировать из D:\xray_engine\sdk\components\ElPack\bpl
...
elpackB6.BPL
elpkdbB6.BPL
elpproB6.BPL
...
в windows\SysWOW64 или в windows\system32 (если x86)

***

*1 - первоисточник: http://files.xray-engine.org/files/xray_engine/src/soc_cs/
Архив: stk-src-tags.7z
Папка: stalker patch 1.0004 (30.07.2007)

*********************************************************************

Смонтировать диски: D:\xray_engine\disk_i_x_mount.cmd

Запустить Visual Studio, в Tools -> Options -> Projects and Solutions -> VC++ Directories -> Show directories for:

добавить в include files:

X:\sdk\boost_1_30_0
X:\sdk\dxsdk9\include
X:\sdk\eax\include
X:\sdk\FreeImage
X:\sdk\loki
X:\sdk\luabind\luabind
X:\sdk\MagicSoftware\Sdk\Include
X:\sdk\OpenAL\Include
X:\sdk\Xiph\Include
X:\xray-svn-trunk\xrCore
X:\xray-svn-trunk\xrLua\src


добавить в library files:

X:\sdk\dxsdk9\lib
X:\sdk\eax
X:\sdk\FreeImage
X:\sdk\MagicSoftware\Sdk\Library\ReleaseDLL
X:\sdk\OpenAL\libs
X:\sdk\Xiph\Libs
X:\libraries

*********************************************************************

После, открыть D:\xray_engine\xray-svn-trunk\xr_3da\XR_3DA.sln

Собрать нужные / все проекты. Для сборки сдк требуется собрать на Mixed-конфиге:
...
ETools
xrCore
xrSE_Factory
и те, что требуются для этих 3-х

(проверено: на релизном конфиге, собранный xrAI компилит сетку без укрытий, в mixed норм)

*********************************************************************

Запустить C++ Builder 6, в меню Tools -> Environment Options -> Environment Variables -> User Overrides -> New...

добавить:

AL_DIR = D:\xray_engine\sdk\openal
COMPONENTS = D:\xray_engine\sdk\components
CORE_DIR = D:\xray_engine\xray-svn-trunk\xrCore
DXSDK9_DIR = D:\xray_engine\sdk\dxsdk9\include
EAX_DIR = D:\xray_engine\sdk\eax
ETOOLS_DIR = D:\xray_engine\xray-svn-trunk\ETools
MAGIC_SW_DIR = D:\xray_engine\sdk\magicsoftware\freemagic\include
SDK_DIR = D:\xray_engine\sdk
XIPH_DIR = D:\xray_engine\sdk\xiph\include


в меню Component -> Install Packages -> Add...

D:\xray_engine\sdk\components\AlexMX\editor.bpl
D:\xray_engine\sdk\components\ElPack\Code\dceldbB6.bpl
D:\xray_engine\sdk\components\ElPack\Code\dcelppB6.bpl
D:\xray_engine\sdk\components\ElPack\Code\dclelpB6.bpl

*********************************************************************

После, открыть D:\xray_engine\xray-svn-trunk\all_editors.bpg и собрать проекты.

xrCoreB
xrSoundB
xrParticlesB
xrEPropsB
xrECoreB
